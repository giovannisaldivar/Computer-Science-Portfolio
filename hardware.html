Hardware

Motherboard

The motherboard is the main circuit board. It contains all the buses found in a computer. These buses allow data to travel between the various components in a computer. A motherboard is also known as the system board, the backplane, or the main board. The chip set is one of the most important components in the mother board. It allows the CPU to communicate. .

CPU

The CPU (Central Processing Unit) is in charge of processing data. How it will process data will depend on the program. The program can be anything. But for the CPU it makes no difference. It just follows the orders contained inside the program.

RAM


RAM stands for Random Access Memory. It takes things from the hard drive and stores it for faster access. For example, if you are playing a game the computer takes all the textures and puts it on the RAM when the game starts and loads programs faster.

Hard Drive


Storage is the hard drive's responsibility. Everything you keep on your computer is on a hard drive like pictures, music, videos, programs, your preferences, even your operating system they're all stored on your computer's hard drive.

Keyboard


A keyboard is a typewriter-style device, which uses an arrangement of keys, to act as switches to write a certain number or letter. It is used to input text, characters and other commands into a computer. Christopher Latham Sholes was an American inventor who invented the first practical typewriter.

Mouse


A mouse is a pointing device that detects motion relative to a surface. Which allows for fine control of a graphical user interface. It contains at least three buttons. The right and left click and the scroll wheel. They are used to click on certain things on the computer monitor. The scrll wheel is able to scroll through long webpages fastly and easily. Some mice also have extra buttons and cooler functiions than the standard mouse.

Monitor


A monitor is used to provide visual output from a computer. It allows you to see anything that it is connected to your computer. It displays anything that your computer tells it to. It can displays your work you are doing or your game that you are playing. It also shows where your cursor is.

Bus


A bus is a communication system that transfers data between components inside a computer, or between computers. It is how all the different components in a computer with one another.
