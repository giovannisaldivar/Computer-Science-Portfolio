HTML!

What is HTML and Why is it Important?

HTML stands for Hypertext Markup Language. It is used as the standard markup language used to create web pages. Tim Berners-Lee was the creator of HTML and was created in 1999. The language is written in the form of HTML elements consisting of tags enclosed in angle brackets. Browsers do not display the HTML tags and scripts, but use them to interpret the content of the page.

What is CSS and Why is it Important?

CSS stands for Cascading Style Sheet. CSS can be added to individual HTML elements as the value of the style attribute. This method is called inline CSS. CSS was developed to provide style sheets for the web. One requirement for a web style sheet language was for style sheets to come from different sources on the web. CSS is a style sheet language used for describing the look and formatting of a document written in a markup language.

How do Tags Work?

HTML tags are the hidden keywords within a web page that define how the browser must format and display the content. A tag is written by adding a (<) and a (>) to an element that is put inside them.

What are Attributes and How do They Work?

To change the attributes you use the by changing the attribute you want. For example, the id, title, class, and style are all attributes you can change.

Description of at Least Four Different Tags

The 'p' tag is use to make a paragraph. The 'style' tag is used to customize your page and make it look the way you want it to. The 'br' tag is used to add a break in a text if you want to seperate song lyrics just add the 'br' tag where you want it to break. The 'h1, h2, h3, h4, h5, h6' tags are used to make a heading but the smaller the number the bigger the text will be on the page. So the 'h1' tag has the biggest heading font and the 'h6' tag has the smallest heading font size.
