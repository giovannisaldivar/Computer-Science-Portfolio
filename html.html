HTML

HTML stands for Hypertext Markup Language. It is used to make web pages. Tim Berners-Lee was the creator of HTML and was created in 1989. The language is written in the form of HTML elements consisting of tags enclosed in angle brackets. HTML is like the skeleton of the website. You tell it what to do and show and it will complete that order in the website when the viewer is loking at it. HTML has improved since it was invented. There has been many updates with HTML to make it even better than before. There has been over ten different updates for HTML! HTML is made up of many different sections. For example, some of them are elements, tags, and attributes, but there are many more.

CSS is also used to move all the borders, font, colors, backgrounds, etc. to another file so you can make your HTML file much more simple.

Tags

HTML tags are like commands, anything you put in the angle brackets will command the webpage to do it. A tag is written by adding a (<) and a (>) to an element that is put inside them. For example, the p, h1, body, a, style, and head tags are some examples of tags, but there are many more tags than just these. Each tag is made to do something. The p tag is made to make a paragraph.

Attributes

HTML attributes are modifiers that modify an element type. Some HTML attributes are the href attribute, the id attribute, and the title attribute. There are many more attributes! To change the attributes you add or change the value of the attribute.

Description of at Least Four Different Tags

The 'p' tag is use to make a paragraph. The 'style' tag is used to customize your page and make it look the way you want it to. The 'br' tag is used to add a break in a text if you want to seperate song lyrics just add the 'br' tag where you want it to break. The 'h1, h2, h3, h4, h5, h6' tags are used to make a heading but the smaller the number the bigger the text will be on the page. So the 'h1' tag has the biggest heading font and the 'h6' tag has the smallest heading font size.
